# WordleSolver
## 실행
wordle.cpp 파일 디렉토리에서 다음을 실행
> g++ wordle.cpp -o wordle
> 
> ./wordle
## 사용
추천되는 단어를 사이트에 입력 후 결과물에 대해 다음과 같이 프로그램에 입력
- 0:miss
- 1:ball
- 2:strike



ex)
- given word : ulcer
- our input  : bales
- cmd input  : 00120
